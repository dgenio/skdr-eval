`evaluate_sklearn_models` gained an `n_jobs` parameter that parallelises three independent axes — the candidate-model loop, the propensity/outcome cross-fitting folds, and the bootstrap replicates (`ci_bootstrap=True`). Results are deterministic and independent of `n_jobs`: the model loop and folds are bit-identical to serial, and the bootstrap reseeds per replicate from a `SeedSequence` so a parallel run reproduces the serial CIs exactly. `joblib` is now a core dependency (already transitive via scikit-learn).
